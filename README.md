##$Tujuan dari analisis ini adalah untuk membandingkan algoritma machine learning(klasifikasi) terbaik untuk mengklasifikasikan penyakit stroke
Data terdiri dari 12 kolom :
- id
- gender 
- age
- hypertension
- heart_disease 
- ever_married 
- work_type
- Residence_type
- avg_glucose_level
- bmi
- smoking_status
- stroke
###Algoritma yang digunakan :
- k-NN
- Naive Bayes
